from datetime import timezone

now_utc = datetime.now(timezone.utc)
entry_ts = shadow.entry_timestamp
if entry_ts and shadow.timeout_candles:

## FIX B3 — Vocabulário direction consistente

### Vocabulário correto definido

```
'SPOT'  → compra spot, long-only, sem alavancagem
          TP: preço >= tp_price
          SL: preço <= sl_price

'LONG'  → futures comprado (uso futuro)
          TP: preço >= tp_price
          SL: preço <= sl_price

'SHORT' → futures vendido (uso futuro)
          TP: preço <= tp_price  ← invertido
          SL: preço >= sl_price  ← invertido
```

### Fix em _create_from_decision

Localizar a linha com `"direction": "long"` e substituir por:

```python
"direction": getattr(decision, "direction", None) or "SPOT",
```

Herda o direction da decisions_log. Fallback para 'SPOT'
se NULL (515 registros antigos sem direction).

### Fix na lógica de fechamento

No monitor, preparar para os três tipos (additive):

```python
# FIX B3 — Lógica de fechamento por direction
direction = (shadow.direction or "SPOT").upper()

if direction in ("SPOT", "LONG"):
    tp_hit = current_price >= shadow.tp_price
    sl_hit = current_price <= shadow.sl_price
elif direction == "SHORT":
    tp_hit = current_price <= shadow.tp_price
    sl_hit = current_price >= shadow.sl_price
else:
    # Fallback seguro para direction desconhecido
    tp_hit = current_price >= shadow.tp_price
    sl_hit = current_price <= shadow.sl_price
```
