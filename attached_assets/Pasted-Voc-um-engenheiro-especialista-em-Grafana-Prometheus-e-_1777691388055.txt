# 🔹 1. TOP STATS (Stat Panels)

Criar 6 painéis:

1. STATUS DO SISTEMA (calculado)

* OK se:

  * avg(confidence) > 0.6
  * NO_DATA < 20%
  * error_rate < 5%
* Caso contrário: WARNING / CRITICAL

---

2. SCORE MÉDIO

PromQL:
avg(score)

---

3. CONFIDENCE MÉDIA

PromQL:
avg(indicator_confidence)

---

4. DATA QUALITY

SQL (Postgres):

SELECT
100 * COUNT(*) FILTER (WHERE indicators_json->'taker_ratio'->>'status' = 'VALID')
/ COUNT(*)
FROM indicator_snapshots
WHERE timestamp > NOW() - INTERVAL '1 hour';

---

5. REJECTION RATE

PromQL:

sum(rate(score_rejection_total[5m]))
/
sum(rate(total_trades[5m]))

---

6. TRADES (1h)

SQL:

SELECT decision, COUNT(*)
FROM decisions_log
WHERE created_at > NOW() - INTERVAL '1 hour'
GROUP BY decision;

# 🔹 2. DATA QUALITY GAUGES

Criar gauges:

* VALID
* NO_DATA
* STALE
* ERROR

SQL base:

SELECT
COUNT(*) FILTER (WHERE status = 'VALID'),
COUNT(*) FILTER (WHERE status = 'NO_DATA'),
COUNT(*) FILTER (WHERE status = 'STALE'),
COUNT(*) FILTER (WHERE status = 'ERROR')
FROM (
SELECT jsonb_each(indicators_json)
FROM indicator_snapshots
) t;

SQL:

symbol,
key AS indicator,

value->>'status',

FROM indicator_snapshots,
