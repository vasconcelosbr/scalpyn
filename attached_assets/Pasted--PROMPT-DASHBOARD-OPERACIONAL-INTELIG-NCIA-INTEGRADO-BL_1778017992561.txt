## 1. 🟢 HEALTH DO PIPELINE (CRÍTICO)

### Endpoint:

```text id="0ns1s6"
GET /api/dashboard/health
```

### Query:

```sql id="vlybzn"
SELECT 
  COUNT(*) AS rows_5m,
  MAX(time) AS last_candle,
  NOW() - MAX(time) AS delay,
  COUNT(DISTINCT symbol) AS symbols
FROM ohlcv
WHERE timeframe = '5m'
  AND time > NOW() - INTERVAL '15 minutes';
```

### UI:

* indicador verde/amarelo/vermelho
* texto:

```text id="4clpfu"
"Pipeline saudável" / "Atrasado" / "Parado"
```
