2) Guard de completude (mesma regra em todos os caminhos)

Extraia o guard para um módulo comum e use nos três caminhos (pipeline_scan, evaluate_signals, execute_buy):

# app/services/indicator_validity.py

def unwrap(v):
    return v.get("value") if isinstance(v, dict) else v

def is_complete(indicators: dict) -> bool:
    for k in ("rsi", "macd_histogram", "adx"):
        if unwrap(indicators.get(k)) is None:
            return False
    return True
Uso (exemplo)
ind = merged[symbol]

if not is_complete(ind):
    # mesma política em todos:
    return SKIP_INCOMPLETE_DATA
