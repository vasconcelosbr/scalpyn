## ❌ comportamento atual (ERRADO)


## ✔ NOVO COMPORTAMENTO

* NUNCA parar
* continuar tentando indefinidamente
* log CRITICAL a cada múltiplo de 20 tentativas

## Implementar:

```python id="fix1"
async def _run_with_backoff(self, market: str, coro_factory: Callable) -> None:
    attempt = 0
    delay = RECONNECT_BASE_DELAY

    while self._running:
        try:
            await coro_factory()

            if not self._running:
                break

            logger.warning(
                "[%s] WS exited cleanly but still running — reconnecting",
                market,
            )
            attempt = 0
            delay = RECONNECT_BASE_DELAY

        except asyncio.CancelledError:
            return

        except Exception as exc:
            attempt += 1

            if attempt % RECONNECT_MAX_RETRIES == 0:
                logger.critical(
                    "[%s] WS failed %d times — STILL RETRYING. Last error: %s",
                    market, attempt, exc,
                )
            else:
                logger.warning(
                    "[%s] WS lost (attempt %d): %s — retry in %.1fs",
                    market, attempt, exc, delay,
                )

        if not self._running:
            break

        await asyncio.sleep(delay)
        delay = min(delay * 2, RECONNECT_MAX_DELAY)
```
